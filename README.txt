Usage instructions:
From the forge prompt, run the "errai setup" command to install the ErraiFacet.  Installing the ErraiFacet installs the required pom.xml, and sets up project.

Once the Facet is installed, you are ready to develop your Errai based application.

To install a simple example errai-bus-demo, run the "errai install-errai-bus-example" command.
